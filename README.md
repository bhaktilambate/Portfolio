# Portfolio

A responsive personal portfolio website built using HTML, CSS, and JavaScript. The site is designed as a static frontend project and deployed through GitHub Pages.

## Project Overview

This portfolio showcases a clean, professional frontend implementation with a dark cherry-red visual theme, cream typography, animated UI transitions, and structured content sections. It is built without any frontend framework, making it lightweight, fast, and easy to host.

## Technical Features

- Static website architecture
- Responsive layout for desktop and mobile screens
- Custom CSS theme using reusable color variables
- Animated page intro sequence
- Scroll-based reveal animations using Intersection Observer
- Canvas-based grain texture effect
- Decorative background architecture elements
- Hover interactions on project and content cards
- Accessible navigation structure
- SEO and social sharing metadata
- GitHub Pages compatible file structure

## Tech Stack

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Deployment

- This site is deployed using GitHub Pages from the main branch and root directory.
